# 📊 Aadhar Enrollment Dashboard (Power BI)

## 🔍 Overview
This project focuses on analyzing Aadhar enrollment data using Power BI. The dataset was cleaned and preprocessed using Python to ensure accuracy.

## ⚙️ Tools Used
- Power BI
- Python (Pandas, NumPy)

## 📈 Key Features
- State-wise and region-wise enrollment analysis
- Interactive dashboard with slicers and filters
- Data cleaning and preprocessing using Python
- Identification of trends and patterns

## 📊 Insights
- Compared enrollment across different regions
- Identified patterns in Aadhar registrations
- Enabled better understanding through visual dashboards
